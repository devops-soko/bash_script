## Title
check string's compositions and length

## Description
check if string consists of numbers and the length if string is 4 or 6

## I/O Example
| num | return |
|:--------|:--------|
| 1234 | true | 
| a234 | false |
| 123456 | true | 
| a12345 | false |
| 12345 | false |
