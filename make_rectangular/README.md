## Title
Print rectangular by $

## Description
When user input 2 integers, n and m
Print a rectangular with width n and height m using '$'

## Restriction Conditions
n and m are natural numbers, 

1 <= n <= 100

1 <= m <= 100

## I/O Example
input
```
5 3
```
output
```
$$$$$
$$$$$
$$$$$
```

## Environment
OS Version : CentOS 8.3.2011
