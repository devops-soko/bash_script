## Title
get middle letters from string

## Description
Return the middle letter of the input string

## Restriction Conditions
If the length of the word is even, return the middle two letters.

## I/O Example
| input | result |
|:--------|:--------|
| "abcde" | 'c' | 
| "qwer" | 'we' | 


## Environment
OS Version : CentOS 8.3.2011
