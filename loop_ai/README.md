## Title
Print a and i repeatly

## Description
Return a string that the length is n and the pattern is like 'aiaiaiaia.....'.

## Restriction Conditions
n is a natural number of more than 1 and less than 10000.

## I/O Example
| n | result |
|:--------|:--------|
| 3 | 'aia' | 
| 4 | 'aiai' | 


## Environment
OS Version : CentOS 8.3.2011
