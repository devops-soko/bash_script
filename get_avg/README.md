## Title
get average

## Description
Return a string that the length is n and the pattern is like 'aiaiaiaia.....'.

## Restriction Conditions
return the average of input numbers

each integer n is 
0 <= n <= 1000

## I/O Example
| input | result |
|:--------|:--------|
| 1,2,3,4 | 2.500 | 
| 5,5 | 5.000 | 


## Environment
OS Version : CentOS 8.3.2011
