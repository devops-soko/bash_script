## Title
get average

## Description
return the average of input numbers

## Restriction Conditions
maximum number of integers that can be input is 10

each integer n is 
0 <= n <= 1000

## I/O Example
| input | result |
|:--------|:--------|
| 1,2,3,4 | 2.500 | 
| 5,5 | 5.000 | 


## Environment
OS Version : CentOS 8.3.2011
